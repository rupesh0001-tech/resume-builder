# AI Powered MERN Resume Builder

An AI-powered resume builder where users can create resumes using different templates and color themes in seconds.

Live demo: https://resume-builder-delta-black-55.vercel.app/

---

## Features

- AI-generated professional summary using Gemini API
- Dynamic resume builder with live preview
- Multiple resume templates
- Customizable colors and layout
- User authentication (signup / login)
- Save and edit resume data
- Download resume as PDF
- Responsive UI (works on desktop and mobile)

---

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI:** Google Gemini APIs
- **Others:** JWT-based authentication (for protected routes / user sessions)
